Adatbázis rendszerek

Oktató: Dr. Kovács László
Kredit: 5

Tematika

  • Adatkezelés alapfogalmai, adatstruktúra alapok, Adatperzisztencia szintjei: file, táblázatok és adatbázisok. C file kezelés,Táblázatkezelő alapú perzisztencia, Excelben.
  • VBA programozás alapjai Excelben; VBA alkalmazások OOP modellje. Excel adatkezelés haladó programozása, adatkezelési algoritmusok. Algoritmusok hatékonyság elemzése.
  • Az adatbáziskezelés alapfogalmai; az adatbázis rendszer DB fejlesztés lépései, A szemantikai modellek áttekintése, ER adatmodell, EER és UML adatmodell
  • A relációs adatmodell áttekointése, A relációs struktúra (mező, reláció, DB) és integritási feltételek bemutatása. Az ER modell konverziója relációs modellre. SQL DDL elemei
  • A relációs adatmodell műveleti része, relációs algebra elemei: szelekció, projekció, join, aggregáció, csoportképzés, kiterjesztés, halmazműveletek SQL DML és DQL alapok.
  • SQL SELECT utasítás alapelemei elemei, kapcsolódás a relációs algebrához, az SQL SELECT szintaktikája. a SELECT utasítás további elemei. al-SELECT kezelése
  • Az SQL további elemei: speciális függvények adátum kezelésre, sztring kezelésre. Karakter kódkészletek.Tranzakció elemei, VIEW mechanizmus működése és parancsai
  • Az adatmodellezés problémái, anomáliák áttekintése. A mezők közötti függőségek szerepe. Normálformák. Dekompozíciós szabályok, normalizálás lépései
  • Az adatbáziskezelők fizikai architektúrája. A DBMS rendszerek működése: A fizikai szintű tárolási módszerek, indexelés áttekintése. B-fa index működése.
  • Adatbázisok védelmi modelljei, A DAC védelmi modell, szerepkörök alkalmazása; a MAC védelmi modell elemei. SQL védelmi parancsok használata
  • Esettanulmány: MySQL. .A mySQL RDBMS architektúra elemei, adminisztrációs alapok, mySQL SQL áttekintése, alap SQL parancsok használatának bemutatása
  • Esettanulmány: SQLLite. Az SQLite DBMS architektúra elemei, adminisztrációs alapok, SQLite SQL áttekintése
  • Esettanulmány: Oracle APEX keretrendszer. Az APEX DBMS architektúra elemei, adminisztrációs alapok, APEX SQL áttekintése
